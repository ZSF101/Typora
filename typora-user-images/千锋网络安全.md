# 千锋网络安全

## 虚拟机运行机构

1.寄居架构

2.原生架构 ：公司生产环境使用

 



寄居架构（实验环境，测试环境）

- 1.虚拟机作为应用软件安装在操作系统上
- 2.可以在此应用软件上安装多个操作系统
- 3。直接安装在硬件上的系统为宿主

![image-20230729161340536](assets/image-20230729161340536.png)



原生架构（生产环境）

- 1.虚拟机软件直接安装在计算机硬件上
- 2.虚拟机本省就是一个操作系统
- 在这个虚拟机中可以同时运行多个操作系统
